# CS170Project

## How to check the output is valid

1. clone this repo

   ```
   git clone https://github.com/Yao-Shao/CS170Project.git
   ```

2. in terminal

   ```
   cd <path to the utils folder>
   pip3 install requirments.txt
   python3 output_validator.py --all ../inputs ../outputs
   ```
